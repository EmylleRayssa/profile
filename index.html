Um pouco sobre mim

Sou estudante de Ciência da Computação, com uma paixão crescente por banco de dados e inteligência artificial. Estou explorando como essas áreas podem transformar dados em soluções inteligentes e impactantes, sempre buscando unir teoria e prática no desenvolvimento de projetos. Meu foco atual é no desenvolvimento backend com Python, aliado ao estudo de estruturas de dados e algoritmos voltados para IA e gerenciamento eficiente de bancos de dados. Também possuo conhecimentos em JavaScript, HTML e CSS, criando aplicações completas e funcionais. Curiosa e dedicada, acredito que a tecnologia tem o poder de resolver desafios complexos e criar oportunidades. Estou sempre em busca de aprender, experimentar e crescer como profissional em um campo que está em constante evolução.

Eu faço coisas incríveis para os clientes

Cientista de Dados

Extrair dados de APIs, bancos de dados ou realizar web scraping, Remover duplicatas, tratar outliers e preencher valores ausentes, Criar gráficos, tabelas e cálculos estatísticos para compreender os dados, Construir modelos para prever vendas futuras, identificar fraudes ou recomendar produtos, Criar dashboards interativos, relatórios visuais e apresentações, Configurar scripts automatizados para rodar análises recorrentes, Sugerir mudanças no negócio para reduzir custos ou melhorar eficiência, Implementar checks automáticos para evitar problemas de qualidade nos dados analisados e Criar ferramentas que integrem dados ao sistema de uma empresa.

Web Designer

Planejar onde menus, imagens e textos estarão posicionados, Escolher cores, fontes, ícones e botões que melhorem a aparência e a usabilidade, Organizar informações para facilitar a navegação e reduzir o tempo para encontrar o que o usuário procura, Ajustar o layout para que o site fique bonito e funcional em diferentes tamanhos de tela, Adicionar banners animados ou sliders de imagens, Escrever códigos para estilizar botões, ajustar margens ou adicionar interatividade simples, Reduzir o tamanho de imagens para melhorar o tempo de carregamento do site e Testar o site em navegadores e dispositivos diferentes para garantir que funciona corretamente

Analista de BI

Integrar dados financeiros de um sistema ERP com dados de vendas de um CRM, Criar dashboards em ferramentas como Power BI ou Tableau para monitorar o desempenho de vendas, Detectar uma queda nas vendas em uma região específica e investigar as causas, Medir o churn de clientes ou o tempo médio de atendimento, Criar um esquema de data warehouse para consolidar dados de diferentes áreas da empresa e Configurar scripts para gerar relatórios automáticos semanalmente
